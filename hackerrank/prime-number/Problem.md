# Prime Number

Write a program to find whether a given number is prime or not.  
A prime number has exactly two factors: 1 and itself.  

---

### Function Signature

```java
String primeNumber(int n)
```

### Parameters

- `int n`: the number to test for primality

### Returns

- `String`: `"yes"` if `n` is prime, otherwise `"no"`

### Input Format

- A single line containing the integer `n`.

### Constraints

- `1 ≤ n ≤ 10^9`

### Sample Input

```
11
```

### Sample Output

```
yes
```

### Explanation

11 is divisible only by 1 and 11, so it’s prime.
